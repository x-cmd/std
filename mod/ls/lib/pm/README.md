# 目标

1. 内置常用的plugin: yank -- 以笔记本触摸板方式为目标
2. tpm用来给更广泛和更垂直的定制
3. 支持一个window一个主题
4. 支持本地和远程主题

需求来源要控制：

1. 如果我们要和 prometheous 之类的大型监控应用对比，那是一场ROT极低而且胜率极低的战争。
2. 我们唯一的优势是轻量级，以及灵活，可编程
3. 要做到上述优势，最重要的事情是各模块和UI组件

因为3，我们不可能马上上线，这将是V1的silver bullet。
那我们能做什么？

1. tpm内置支持
2. tmux推广
3. workspace功能
4. 内部用来看团队内部信息
5. tapp
